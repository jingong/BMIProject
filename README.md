# BMIProject
身高体重指数测量系统

前端采用HBuilder<br>
后端采用IDEA<br>
数据库采用MySQL<br>

主要功能：<br>
1.输入身高和体重计算BMI值<br>
2.计算的结果写入数据库，并追加到历史记录里<br>
3.删除查询记录<br>
